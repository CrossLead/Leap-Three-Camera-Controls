Paddle Controls

Paddle Controls are a type of control that allows the user to 'paddle' through a scene. If a user 'slices' through the field, the camera will not move, but if they 'paddle' through it, it will. The parameters are as follows:

  • Dampening: This will alter how fast the field will slow down
  • Direction Strength: This will alter how much the hand must be 'paddling.
  • Power: This will alter the falloff for movement.
  • Division Factor: This is a divides the speed, and is well paired with power
  • Max Velocity: This makes sure that the speed doesn't go over a certain amount.

Trackball Controls

Trackball Controls enable the user to use the scene like a giant ball that they push around with a single hand. Although there is a 'zoom' part of this controls, currently it is based more on rotation than it is is translation. The parameters as follows:

  • Rotation Speed: Changes how fast the scene will rotate.
  • Rotation Dampening: Changes how fast the scene will slow down
  • Zoom Dampening: Changes how fast the Zoom will slow down
  • Min Zoom: The minimum distance the camera can get from the center
  • Max Zoom: The maxium distance the camera can get from the center

Two Hand Controls

Two Hand Controls let a user move around the scene by pinching. Once engaged in a pinch the user can simply move their hand to move the scene, or additionally begin pinching with the second hand to rotate, scale, or translate the scene. The parameters are as follows:

  • Translation Speed: Speed that you move the scene when you pinch
  • Translation Decay: How the scene slows down when you let go
  • Scale Decay: How the scene scales down when you let go
  • Rotation Slerp: How the rotation updates when you rotate
  • Rotation Speed: How fast you rotate
  • Pinch Threshold: What point a pinch starts
  • Smoothin Value: Rate at which the data gets smoothed.

Spring Controls

Spring Controls use physics to create a interaction method. By pinching, the user places an 'anchor' at the position they pinched. A spring is connected between the camera and the anchor, so it gets pulled to a stable position a certain distance away from the anchor. Users can also keep their hand pinched to smoothly move / look around the scene. The parameters are as follows:

  • Dampening: How fast the scene slows down
  • Size: How far away the hand can move.
  • Spring Constant: Variable to define physics of spring
  • Static Length: Resting distance from target
  • Mass: Variable to define physics of spring
  • Anchor to Target: Rate at which the anchor tweens to the target
